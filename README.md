# The Weeknd

Pagina web dedicada a mi artista favorito, The Weeknd (Abel Tesfaye). Creada como proyecto de formacion del programa ADSO del SENA.

## Contenido

- **Inicio** — Landing page con video destacado y enlaces a las secciones.
- **Biografia** — Historia y datos de Abel Tesfaye.
- **Albumes** — Discografia completa con sus albumes mas importantes.
- **Playlist** — Lista de canciones esenciales con album y duracion.
- **Galeria** — Imagenes del artista y portadas de albumes.

## Tecnologias

- HTML5
- CSS3
- JavaScript
- Libreria AOS (animaciones al hacer scroll)
